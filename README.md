# HoliJob v2

## 프로젝트 개요
- **이름**: HoliJob (홀리잡)
- **핵심 가치**: "일자리부터 골라드릴게요" - 사용자가 생각하지 않고도 결정 → 지원 → 준비가 자동으로 흐르는 서비스
- **목표**: 워킹홀리데이 출국 전에 AI가 자동으로 최적의 일자리를 매칭하고 원터치로 지원

## 주요 개선사항 (v2)

### ✨ 핵심 로직 개선
1. **상태 기반 플로우**
   - `user_status`: NEW → BASIC_READY → PROFILE_COMPLETE
   - `match_status`: NONE → MATCHING → READY_TO_APPLY → APPLIED → PRE_INTERVIEW
   - `application_status`: NONE → SUBMITTED → VIEWED → REQUEST_INFO → REJECTED

2. **홈 화면 조건 분기**
   - 매칭 중: "당신 조건에 맞는 일자리 찾는 중"
   - 지원 준비: 1순위 카드 + 원터치 지원 버튼
   - 지원 완료: 진행 상태 및 다음 단계 안내
   - 인터뷰 직전: 인터뷰 준비 가이드

3. **원터치 지원 로직**
   - 프로필 정보 자동 체크
   - 정보 부족 시 즉시 보완 모달 표시
   - 한 번만 입력하면 다음부턴 진짜 원터치

4. **정보 보완 모달**
   - 필요한 정보만 요청 (영문 이름, 국적, 여권)
   - 여권 OCR 우선 제안
   - 저장 후 자동으로 지원 완료

5. **이탈 복구**
   - LocalStorage 자동 저장
   - 재진입 시 "이어서 할까요?" 제안
   - 상태에 따라 적절한 화면으로 복귀

### 🎯 주요 화면

#### 1. 스플래시
- 심플한 로고와 "일자리부터 골라드릴게요" 메시지
- [시작하기] 버튼

#### 2. 온보딩
- **3가지 핵심 질문만**
  - 어디로 가시나요? (국가 + 도시)
  - 출국일
  - 일 시작 가능일
- 입력 자동 저장
- 날짜 유효성 검증

#### 3. AI 매칭
- 2초 로딩 애니메이션
- 진행 상황 실시간 표시
  - ✓ Sydney 지역 일자리 검색 중
  - ✓ 시작일 5월 15일 매칭
  - ⏳ 최적의 후보 선정 중

#### 4. 홈 (핵심 화면)
- **상태별 다른 UI**
  - 매칭 중: 로딩 상태
  - 지원 준비: 1순위 카드 (95% 매칭 스코어 표시)
  - 지원 완료: 진행 상태 트래킹
- **1순위 카드 상세**
  - 매칭 스코어 강조
  - 직무, 고용주, 위치, 시급
  - 선정 이유 박스
  - 주요 혜택 태그
  - 큰 원터치 지원 버튼
- **퀵 통계**
  - 체크리스트 진행률
  - 출국까지 D-day

#### 5. 일자리 탭
- 2~3순위 후보들
- 각 카드마다 매칭 스코어 표시
- 개별 원터치 지원 가능
- 하단에 "직접 검색하기" 링크 (보조 경로)

#### 6. 체크리스트 탭
- 출국일 기준 우선순위 자동 조정
  - D-30 이내: 출국 전 체크리스트 우선
  - D-30 이상: 근무 전 체크리스트 우선
- 자동화 가능 항목 표시
- 진행률 바

#### 7. 마이페이지
- 프로필 완성도 (0-100%)
- 기본 정보 (목적지, 날짜)
- 프로필 정보 (이름, 국적, 여권)
- Services Hub
  - 일자리 매칭: 활성화
  - 항공권: 준비중
  - 숙박: 준비중
  - 비자: 준비중

#### 8. 정보 보완 모달
- 깔끔한 모달 UI
- 부족한 정보만 요청
- 여권 사진 자동 입력 버튼
- [저장하고 지원하기] / [나중에]

#### 9. 지원 완료
- 애니메이션 효과
- 지원한 일자리 요약
- 다음 단계 안내
- [홈으로 가기] / [다른 일자리 보기]

## 주요 플로우

### 신규 사용자
```
[스플래시] → [온보딩 3문항] → [AI 매칭 2초] → [홈 1순위 카드]
→ [원터치 지원] → (정보 부족 시) [모달 입력] → [지원 완료]
```

### 복귀 사용자
```
[스플래시] → "이어서 할까요?" → [홈 (현재 상태)]
```

### 지원 후
```
[홈] → 지원 완료 상태 표시 → 고용주 검토 대기
→ (24-48시간) → 추가 정보 요청 or 인터뷰 제안
```

## 핵심 원칙 (v2)

1. **홈 = 항상 결정 화면**
   - 매칭 상태에 따라 자동으로 적절한 UI 표시
   - 사용자가 "다음에 뭘 해야 하지?" 고민하지 않게

2. **원터치 = 정보 있으면 1클릭**
   - 프로필 완성되면 진짜 원터치
   - 부족하면 그 순간만 보완

3. **자동매칭 = 대신 결정**
   - 추천이 아니라 "이게 최선"
   - 1순위를 크게, 나머지는 보조

4. **직접찾기 = 보조 경로**
   - 기본 플로우를 방해하지 않음
   - 원하는 사람만 접근

5. **상태 저장 = 이탈 방지**
   - 모든 입력 자동 저장
   - 돌아왔을 때 이어하기 제안

## 데모 성공 기준

✅ **5초 안에 이해**
- "아, 이 앱이 일자리를 골라주는구나"

✅ **10초 안에 가치 체험**
- 3문항 입력 → AI 매칭 → 1순위 카드

✅ **원터치의 진짜 의미**
- 정보만 있으면 정말로 1클릭
- 없어도 그 자리에서 바로 보완

✅ **복귀해도 끊기지 않음**
- 입력한 정보 유지
- 상태 기반으로 자연스럽게 이어짐
