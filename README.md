## v5 주요 변경사항

### 📋 체계적인 6단계 온보딩

#### Step 1: 도시 선택
- 목적지 도시 선택
- 5개 옵션: Sydney, Melbourne, Brisbane, Perth, Adelaide

#### Step 2: 출국일 선택
- 날짜 입력
- 대략적인 계획 파악

#### Step 3: 여권 스캔
- 여권 사진면 스캔 (시뮬레이션)
- OCR 자동 입력
  - 이름 (영문)
  - 국적
  - 여권 번호
  - 유효기간

#### Step 4: 성향 파악
- **워홀 경험**
  - 처음이에요
  - 경험이 있어요
- **영어 수준**
  - 초급 (기본 회화)
  - 중급 (업무 가능)
  - 고급 (능숙함)

#### Step 5: 원하는 직업군
- 복수 선택 가능
- **옵션**
  - ☕ 호스피탈리티 (카페, 레스토랑)
  - 🌾 농장/목장 (과일 수확, 목축)
  - 🛍️ 리테일 (매장, 판매)
  - 💼 오피스 (사무, 관리)

#### Step 6: 이력서 업로드 (선택)
- PDF, DOC, JPG 지원
- 건너뛰기 가능
- 나중에 올려도 됨

### 🎯 토스 스타일 특징

#### 한 화면 한 질문
- 집중력 유지
- 명확한 선택
- 부담 없는 진행

#### 진행률 표시
- 상단 "1/6, 2/6..." 표시
- 뒤로가기 버튼
- 단계별 완료 확인

#### 간편한 입력
- 큰 버튼
- 명확한 라벨
- 시각적 피드백

### 📱 온보딩 플로우

```
스플래시
  ↓
Step 1: 도시 (Sydney)
  ↓
Step 2: 날짜 (2024-03-01)
  ↓
Step 3: 여권 스캔 (OCR)
  ↓
Step 4: 성향 (처음/초급)
  ↓
Step 5: 직업군 (호스피탈리티, 리테일)
  ↓
Step 6: 이력서 (선택/건너뛰기)
  ↓
AI 매칭 로딩
  ↓
홈 화면 (1순위 일자리)
```

### 🎯 핵심 기능

#### 1️⃣ 온보딩 (30초)
- 도시 선택 (5개 칩)
- 출국일 입력 (선택)
- AI 매칭 시작 버튼

#### 2️⃣ AI 매칭 로딩
- **수정된 로딩 애니메이션**
- 1초 시뮬레이션
- 진행 상황 표시

#### 3️⃣ 홈 화면
- 1순위 일자리 카드
  - 배지: AI 추천 1순위
  - 매칭 스코어: 95%
  - 직무/도시/시급/시작일
  - 선정 이유
- 원터치 지원 버튼
- 요약 카드 (체크리스트/비자/항공)

#### 4️⃣ 일자리 탭
- 2~3개 후보 카드
- 각 카드별 원터치 지원
- 매칭 스코어 표시

#### 5️⃣ 체크리스트 탭
- 출국 전 준비 (4개)
- 근무 전 준비 (3개)
- 체크박스 터치 최적화

#### 6️⃣ 마이페이지
- 프로필 정보 표시
- 프로필 완성도 진행바
- Services Hub
  - 비자 자동화 (n8n)
  - 항공권 추천 (3옵션)
  - 숙소 추천 (3옵션)

### 🎬 원터치 지원 흐름
1. 지원 버튼 클릭
2. 정보 부족 시 BottomSheet 표시
   - 프로필 입력 (이름/국적)
   - 여권 스캔 (OCR 시뮬레이션)
3. 자동 제출 (0.8초)
4. 완료 토스트
5. 홈 카드 상태 변경
